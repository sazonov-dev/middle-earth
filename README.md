# Игра средиземье (l2-wb)

**Для локального запуска проекта необходимо выполнить следующие команды в терминале:**
```
git clone https://github.com/sazonov-dev/middle-earth
```
```
cd middle-earth
```
```
npm i
```
```
npm run dev
```
